# Authentic Courses

## Features

- Authentication system
- Private route
- Toggle Dark and Light mood
- Download Pdf system
- Active route
- Firebase hosting
- Loging user picture and name show.

## Frameworks and Libraries

- React js
- Tailwind
- Daisy UI
- Mumba UI
- Flowbite
